## ✨ Funcionalidades

| Funcionalidade | Descrição |
|----------------|-----------|
| 👁️ **Mostrar/Ocultar senha** | Clique no ícone de olho para alternar a visibilidade da senha |
| 📱 **Design responsivo** | Adapta-se perfeitamente a dispositivos móveis (até 500px) |
| 🎯 **Foco visual** | Campos ganham contorno verde ao serem selecionados |
| 🖼️ **Ícones intuitivos** | Cada campo possui ícone representativo (usuário, email, etc.) |
| 🎨 **Interface moderna** | Design limpo com sombras suaves e fonte Poppins |
| 📅 **Campo de data nativo** | Seletor de data integrado do navegador |
| 🔘 **Radio buttons estilizados** | Gênero com cores personalizadas |

## 🛠️ Tecnologias utilizadas

- **HTML5** - Estrutura semântica e acessível
- **CSS3** - Flexbox, Grid, Media Queries, variáveis
- **JavaScript (ES6)** - Manipulação do DOM e interatividade
- **Font Awesome 6.7.2** - Biblioteca de ícones
- **Google Fonts** - Fonte Poppins
